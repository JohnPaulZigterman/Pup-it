# Pup-It Product Philosophy

Pup-It is a creation playground disguised as an animation workstation.

The core loop is:

1. Build the space: choose only a rig, then assemble the character, set, and visual identity yourself.
2. Make a weird thing.
3. Perform the weird thing.
4. Turn the weird thing into a show.

## Development Pillars

- Playground first: users should feel invited to make original characters, sets, props, styles, effects, and little behavior systems inside the app.
- Default choices should feel like blank rigging models and creative prompts, not finished stock characters.
- Performance joy: controlling a puppet should feel rewarding before the user understands the deeper toolset.
- Show factory: every good idea should become reusable as cast, sets, style rules, templates, takes, timelines, and export packages.
- Roughness is allowed: the app should encourage handmade, funny, strange, imperfect work instead of sanding everything into one stock look.
- Progressive depth: beginner mode should expose the playful high-impact controls; pro mode can reveal exact tuning, pipeline controls, and advanced systems.

## Current Sequencing

1. Build the playground tools that create originality fast.
2. Strengthen the performance studio so those creations feel good live.
3. Expand the show factory so original work becomes repeatable and airable.
